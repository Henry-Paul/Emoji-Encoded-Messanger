How to Encode
  1. Go to the Encoder tab
  2. Select an emoji from the keyboard
  3. Enter your secret message
  4. Set a password
  5. Copy the emoji and share it
  6. Share the password separately
  7. Tell recipient to use this same app

How to Decode
  1. Go to the Decoder tab
  2. Paste the emoji you received
  3. Enter the password
  4. Click "Decode Message"
  5. The message will appear for 60 seconds
